Blockmap Explorer Mode

This mode lets you view the map's blockmap. Available in Doom and Hexen format
Menu path: Mode -> Blockmap Explorer Mode
Default key: None.

The information about the map's blockmap, and the currently highlighted block will be shown in the mode's docker.

Holding shift will display all shared block of the currently highlighted block.

The docker shows the following info:

Block info
Shows info about the currently highlighted block.

  • Column: the column
  • Row: the row
  • Offset (bytes): the offset of the block list from the start of the BLOCKMAP lump in bytes
  • Is sublist: if the block points to a sublist in the block list, i.e. if the offset points to the first linedef index in the block list or not
  • Lines in block: the number of lines in the block
Blockmap info Shows info about the whole blockmap.
  • Columns: total number of columns
  • Rows: total number of rows
  • Total blocks: total number of blocks (columns * rows)
  • Unique blocks: number of unique blocks (number of total blocks minus the number of blocks saved by reusing blocks)
  • Lines not in blocks: number of lines that are not present in any block
  • Lump size: the size of the BLOCKMAP lump in bytes
  • Offset list end: where the header plus offset list ends in bytes
Questionable offsets
If there are questionable offsets in the offset list a message will be shown. It will also show a checkbox that, when enabled, will highlight all blocks deemed to be quiestionable.
A block is deemed questionable if the offset points inside the header or offset list of the lump.
